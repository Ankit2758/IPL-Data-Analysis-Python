# IPL Data Analysis using Python 🏏

## 📌 Project Overview

This project analyzes IPL (Indian Premier League) data using Python. It includes data cleaning, visualization, and regression modeling to extract meaningful insights.

## 📊 Features

* Top batsman analysis
* Top bowler analysis
* Runs trend by over
* Team performance analysis
* Regression model for prediction

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## 📁 Files

* `project.py` → main code
* `data.csv` → dataset

## 🚀 Output

* Graphs and charts
* Model predictions
* Insights on IPL data

## 📌 Conclusion

This project demonstrates how data analysis can be applied in sports analytics using Python.
